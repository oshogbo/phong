# Usage
    UP/DWON/LEFT/RIGHT/PgDn/PgUp - move light
    a - ambient color
    s - specular color
    d - diffuse color
    b - background color
    p - speed
    - - sb color
    0 - add color
    x - turn on clamp
    c - turn off clamp
    h - show this color
    1 - set changing red color
    2 - set changing green color
    3 - set changing blue color
    4 - set changing all color
    5 - scale mode on
    6 - scale mode off
    o - Kspec
    j - Kamp
    k - Kdiff
    l - alpha
